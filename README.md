# Supplementary Material for UJA-JBHI-2026

This repository provides supplementary material for the article:

**Unsupervised Joint Alignment Framework for Cross-Site Depression Classification Using fMRI**

Authors: Deyi Ren, Youjun Li, Christina Carlisi, and Zi-Gang Huang  
Journal: IEEE Journal of Biomedical and Health Informatics  
DOI: 10.1109/JBHI.2026.3686006

## Contents

- `Supplementary_Competing_Methods.md`: Detailed implementation settings of the competing methods compared in the main article.

## Purpose

The supplementary material in this repository contains detailed methodological descriptions that were moved from the main article to satisfy the page-limit requirement during IEEE proof correction.
